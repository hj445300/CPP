Overview

One of the methods of gauge creation is with C++. This system of programming gauges is ideal for users who want flexibility in creating their gauges but are not familiar with the graphical toolkits supported by Prepar3D. A gauge can be created with C++ alone, or with XML working in tandem. These are called Mixed Mode gauges. This method can be useful to implement XML driven visuals backed by C++ programatic functionality. Graphics for mixed mode and XML gauges can be created with the assistance of the ACE tool, described in the Creating XML Gauges document.

Gauge Tutorials

MODULE_VAR

Each drawing element structure has one or more MODULE_VAR members. You can initialize these members to one of the token variable values. Graphical elements can use the MODULE_VAR structure member to determine the state in which the element will be rendered.
